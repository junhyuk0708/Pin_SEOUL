명시돼 있지 않아 확인이 필요합니다 (담당 교수/코치 또는 데이터 제공팀에 문의 권장).

## 출처 표기 문구 (SOURCE.md 제공, 그대로 사용)

```
이 서비스는 한국관광공사 Tour API(TourAPI 4.0)의 데이터를 활용하였습니다.
출처: 한국관광공사 (https://www.data.go.kr/data/15101578/openapi.do)
라이선스: 공공누리 제3유형
```

이 문구는 서비스 푸터나 "정보" 화면 등에 노출해야 공공누리 3유형의
"출처 표시" 조건을 충족합니다. 팀 전체 확인 항목이라 박지우님 배포 담당과도 공유하는 게 좋습니다.
